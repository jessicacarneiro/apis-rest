## Maven commands
* `./mvnw test`: Run all tests
* `./mvnw verify`: Run all tests and generate JaCoCo report

## JaCoCo report
Run `./mvnw verify` to run all tests and generate a JaCoCo report. The report will be located at `/target/site/jacoco/index.html`
